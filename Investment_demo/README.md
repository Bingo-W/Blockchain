# README

## 选题构思与背景

+ 关于目前的量化投资，仅仅还停留在某个中心机构对于整个项目的投资，像散户并没有所谓的量化。因此该项目致力于将量化进一步延申到投资者自己的手中，决定自己什么时候可以入市跟出市。
+ 拥有健壮的量化模型的人可以发起基金筹集，但达到一定日期为筹集到规定的资金则失败；如果成功则开启量化投资模型。参与投资的人每天在交易时间之外可以进行检查，如果亏损或者盈利达到一定的额度，将会自动结算并返还资金。

